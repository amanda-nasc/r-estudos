# 📊 Estudos em R

Este repositório reúne meus aprendizados em linguagem R, passando pelos principais fundamentos até aplicações práticas voltadas para análise de dados.

## 📚 Conteúdos abordados

- Conceitos básicos da linguagem R  
- Tipos de dados e estruturas  
- Funções  
- Estruturas condicionais e de repetição  
- Manipulação de dados com dplyr  
- Visualização de dados com ggplot2  

## 🎯 Objetivo

Consolidar meu aprendizado em R e construir uma base sólida para projetos de análise de dados e Business Intelligence.

## 📁 Estrutura

Os conteúdos estão organizados por tema e evolução no aprendizado:

- `scripts/`: códigos desenvolvidos ao longo das aulas  
- `projeto-final/`: aplicação prática com análise de dados  

## 🎓 Fonte de estudo

Os conteúdos deste repositório foram desenvolvidos com base em uma playlist de aulas de R disponível no YouTube, produzida pela Hashtag Treinamentos.
